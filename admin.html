📊 ANALYTICS TOKEN

Create a GitHub Fine-Grained PAT with only Gist → Read & Write. This token is saved publicly in site-config.json so all visitors can be tracked.

⚠️ Use a fine-grained PAT scoped only to Gist here — it cannot access your account, repos, or private data.

Create at: github.com/settings/personal-access-tokens/new
→ Resource owner: your account → Repository access: None → Permissions: Gist → Read and write
